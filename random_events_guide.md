# 🎲 隨機事件系統指南

## 系統概述

隨機事件系統為遊戲增加了額外的挑戰和獎勵機會。當玩家點擊"提前開始"按鈕時，有30%機率觸發特殊事件，讓玩家選擇是否挑戰強大的特殊敵人以獲得豐厚獎勵。

## 🎯 觸發條件

- **觸發時機**: 點擊"提前開始 (+50金)"按鈕時
- **觸發機率**: 30%
- **解鎖條件**: 第3波後才會出現
- **選擇權**: 玩家可以選擇接受或拒絕挑戰

## 👹 特殊敵人類型

### 1. 🏆 精英入侵 - 強化版
- **敵人**: 精英戰士
- **特點**: 
  - 高血量 (500 + 波次×100) [強化+25%]
  - 中等速度
  - 25%護甲減傷 [強化+5%]
  - 高傷害 (25 + 波次×1.5) [強化+25%]
  - 八角星外觀，金色光環
- **獎勵**: 150 + 波次×25 金幣 + 傷害加成 (重新平衡)
- **策略**: 強化平衡型敵人，需要更強的綜合防禦

### 2. 🏔️ 巨型威脅 - 強化版
- **敵人**: 巨型怪物
- **特點**:
  - 超高血量 (1000 + 波次×200) [強化+25%]
  - 移動緩慢
  - 50%護甲減傷 [強化+10%]
  - 極高傷害 (35 + 波次×2.5) [強化+17%]
  - 方形外觀，紫色光環
- **獎勵**: 200 + 波次×35 金幣 + 隨機升級 (重新平衡)
- **策略**: 需要更高傷害塔集火，真正考驗火力輸出

### 3. ⚡ 速度惡魔 - 強化版
- **敵人**: 超速刺客
- **特點**:
  - 極高移動速度 (130 + 波次×10) [強化+8%]
  - 30%閃避機率 [強化+5%]
  - 中等血量 (250 + 波次×50) [強化+25%]
  - 中等傷害 (15 + 波次×1.0) [強化+25%]
  - 菱形外觀，青色光環
- **獎勵**: 120 + 波次×20 金幣 + 生命恢復 (重新平衡)
- **策略**: 需要更高射速塔或範圍攻擊，真正考驗命中能力

## 💰 獎勵系統

### 基礎獎勵 (金幣) - 平衡調整
- **精英入侵**: 80 + 波次×15 (降低)
- **巨型威脅**: 100 + 波次×20 (降低)
- **速度惡魔**: 60 + 波次×12 (降低)

### 特殊獎勵 - 平衡調整
- **傷害加成**: 所有塔傷害+15%，持續15秒 (降低)
- **升級折扣**: 下次升級享受50%折扣，30秒內有效 (改為折扣)
- **生命恢復**: 恢復10點生命值 (降低)

## 🎮 戰術建議

### 接受挑戰的時機
✅ **建議接受**:
- 塔陣已經建立完善
- 有足夠的升級投資
- 需要額外資源推進發展
- 特殊獎勵對當前情況有幫助

❌ **建議拒絕**:
- 塔陣還不夠強力
- 剛剛經歷困難波次
- 生命值較低
- 正在存錢準備重要升級

### 針對性策略

#### 對付精英入侵
- **推薦塔**: 狙擊塔 + 機槍塔組合
- **重點**: 平衡傷害輸出，確保穩定擊殺
- **升級**: 優先升級主要輸出塔到2-3級

#### 對付巨型威脅  
- **推薦塔**: 多座高級狙擊塔 + 詛咒塔
- **重點**: 最大化單體傷害輸出
- **升級**: 集中資源升級少數核心塔到滿級

#### 對付速度惡魔
- **推薦塔**: 機槍塔 + 毒氣塔 + 冰凍塔
- **重點**: 高射速和範圍攻擊克制閃避
- **升級**: 提升射速和範圍覆蓋

## 🎊 視覺效果

### 事件開始
- 大量金色粒子爆發效果
- 特殊敵人出現提示

### 特殊敵人外觀
- **精英**: 八角星形狀 + 金色光環
- **巨型**: 大方形 + 紫色光環  
- **速度惡魔**: 菱形 + 青色光環

### 閃避效果
- 速度惡魔閃避時產生白色粒子爆發
- 視覺反饋讓玩家了解攻擊被閃避

### 完成獎勵
- 事件完成時的慶祝粒子效果
- 獎勵獲得的視覺提示

## 📊 風險與收益分析

### 高風險高回報
- 特殊敵人比普通敵人更強大
- 失敗可能導致生命值損失
- 成功獲得的獎勵遠超普通波次

### 策略深度
- 增加了波次間的決策點
- 鼓勵玩家評估自身實力
- 提供額外的資源獲取途徑

### 遊戲節奏
- 打破了固定的波次節奏
- 增加了遊戲的不可預測性
- 提供了更多的戰術選擇

隨機事件系統為遊戲增加了豐富的變化和挑戰，讓每次遊戲都有不同的體驗！合理利用這些機會可以大幅加速遊戲進程。
