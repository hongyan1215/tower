# 塔防遊戲任務系統完整指南 (更新版)

## 任務系統概述

遊戲包含5種不同類型的任務，專注於建造、升級和戰鬥，提供金錢獎勵和特殊效果。任務系統經過簡化，更加直觀易懂。

## 📋 任務類型詳解

### 1. 🏗️ 建造大師
- **任務ID**: `build_towers`
- **描述**: 建造指定數量的塔
- **目標**: 3座 → 5座 → 8座 → 10座 → 15座
- **金錢獎勵**: 150 → 220 → 300 → 450 → 700 (大幅提升)
- **特殊獎勵**: 無 (改為純金錢獎勵)

### 2. ⚔️ 殲滅者
- **任務ID**: `kill_enemies`
- **描述**: 擊殺指定數量的敵人
- **目標**: 20個 → 50個 → 100個 → 200個 → 500個
- **金錢獎勵**: 120 → 180 → 270 → 380 → 600 (大幅提升)
- **特殊獎勵**: 無 (改為純金錢獎勵)

### 3. 🏗️ 專業建造師
- **任務ID**: `build_specific_towers`
- **描述**: 建造指定類型的塔
- **塔類型**: 機槍、狙擊、加農砲、冰凍、連鎖、毒氣、多管、詛咒、金錢、治療
- **目標**: 2座 → 3座 → 5座
- **金錢獎勵**: 180 → 300 → 500 (大幅提升)
- **特殊獎勵**: 無 (改為純金錢獎勵)
- **解鎖條件**: 第2波後才會出現

### 4. ⬆️ 升級專家
- **任務ID**: `upgrade_towers`
- **描述**: 升級指定數量的塔
- **目標**: 2座 → 4座 → 6座 → 8座 → 12座
- **金錢獎勵**: 220 → 380 → 600 → 900 → 1300 (大幅提升)
- **特殊獎勵**: 無 (改為純金錢獎勵)
- **解鎖條件**: 第3波後才會出現

## 🎁 特殊獎勵效果

### 獎勵系統簡化
- **一般任務**: 只給予金錢獎勵，數額大幅提升
- **隨機事件**: 保留特殊獎勵，但強度適中
- **設計理念**: 簡化獎勵系統，避免過度複雜的效果

## 📈 任務策略建議

### 早期任務 (1-3波)
- **優先**: 建造大師、殲滅者
- **策略**: 快速建造塔，積累擊殺數
- **簡單**: 任務目標明確，容易完成

### 中期任務 (4-10波)
- **推薦**: 專業建造師、升級專家
- **策略**: 專精特定塔類型，投資升級
- **重點**: 開始投資升級系統，提升塔的效能

### 後期任務 (11+波)
- **高價值**: 高等級升級專家、專業建造師
- **策略**: 充分利用升級系統，追求效率
- **穩定**: 任務難度適中但獎勵豐厚

### 任務選擇技巧
1. **評估當前實力**: 選擇符合當前塔陣能力的任務
2. **考慮特殊獎勵**: 某些獎勵在特定時機特別有價值
3. **長期規劃**: 某些任務需要多波準備
4. **風險管理**: 不要為了任務而冒險失敗

## 🎮 任務完成技巧

### 建造大師
- 在安全時期快速建造便宜的塔
- 優先選擇機槍塔和冰凍塔（性價比高）

### 殲滅者
- 專注建造高DPS塔
- 機槍塔和狙擊塔是最佳選擇

### 專業建造師
- 根據任務要求建造特定類型塔
- 提前規劃塔的位置和數量
- 了解每種塔的特色和最佳使用場景

### 升級專家
- 優先升級關鍵位置的塔
- 2級升級性價比最高，3級獲得特殊能力
- 平衡升級投資與建造新塔的資源分配

## 🎯 任務系統優勢

### 簡化設計 (進一步優化)
- 移除了複雜的生存、完美防禦和閃電戰任務
- 移除了一般任務的特殊獎勵系統
- 專注於核心遊戲機制：建造、升級、戰鬥
- 任務目標和獎勵都更加直觀明確

### 純金錢獎勵系統
- 所有一般任務只給予金錢獎勵
- 金錢數額大幅提升，補償移除的特殊效果
- 避免複雜的臨時效果和狀態管理
- 讓玩家專注於核心的塔防策略

### 學習引導
- 專業建造師任務幫助玩家了解各種塔的特性
- 升級專家任務鼓勵玩家深入了解升級系統
- 金錢獎勵讓玩家有更多資源嘗試不同策略

### 新增塔類型
- **💰 金錢塔**: 無攻擊能力，定期產生金錢，適合經濟發展
- **💚 治療塔**: 基礎攻擊能力，攻擊敵人同時恢復玩家生命值

任務系統經過簡化後更加友好，專注於核心玩法，讓玩家能夠更好地享受塔防的樂趣！
