# 🎮 粒子塔防戰略遊戲

一個功能豐富的塔防遊戲，採用粒子效果和現代化的遊戲機制。

## ✨ 遊戲特色

- **10種不同類型的防禦塔**：每種塔都有獨特的能力和升級路徑
- **任務系統**：完成任務獲得金錢獎勵，引導遊戲進程
- **隨機事件**：挑戰特殊敵人獲得豐厚獎勵
- **升級系統**：每座塔可升級至3-4級，獲得特殊能力
- **粒子特效**：豐富的視覺效果，提升遊戲體驗
- **動態路徑**：每次遊戲都有不同的敵人路徑

## 🏗️ 防禦塔類型

### 攻擊型塔
- **🔫 機槍塔** (80金) - 快速連射，低傷害，可升級至4級
- **🎯 狙擊塔** (200金) - 超遠射程，高傷害，單體輸出之王
- **💥 加農砲** (180金) - 範圍爆炸傷害，群體清理
- **⚡ 連鎖塔** (220金) - 連鎖閃電攻擊多個目標
- **☠️ 毒氣塔** (140金) - 毒氣雲群體攻擊，降低護甲
- **🎆 多管塔** (250金) - 同時攻擊多個目標

### 輔助型塔
- **❄️ 冰凍塔** (100金) - 減速敵人，控制效果
- **🔮 詛咒塔** (150金) - 增加敵人受到傷害
- **💰 金錢塔** (150金) - 戰術經濟，根據戰況產生金錢
- **💚 治療塔** (160金) - 攻擊敵人並恢復生命值

## 🌟 3級特殊效果

每座塔升級到3級時會獲得特殊能力：

- **機槍塔** - 精準射擊：25%機率雙倍傷害
- **狙擊塔** - 致命一擊：對低血量敵人+80%傷害
- **加農砲** - 強化爆炸：濺射範圍+30%，傷害90%
- **冰凍塔** - 深度冰凍：15%機率完全凍結1秒
- **連鎖塔** - 增強連鎖：連鎖次數+1，範圍+20
- **毒氣塔** - 腐蝕毒素：降護甲20%，持續傷害
- **多管塔** - 追蹤彈藥：子彈追蹤，傷害+20%
- **詛咒塔** - 增強詛咒：易傷效果1.9倍，持續8秒
- **金錢塔** - 投資回報：其他塔收益+20%雙倍獎勵
- **治療塔** - 神聖祝福：25%機率+2HP並減傷50%

## 🎯 遊戲機制

### 任務系統
遊戲包含4種任務類型：
- **建造大師** - 建造指定數量的塔
- **殲滅者** - 擊殺指定數量的敵人
- **專業建造師** - 建造指定類型的塔
- **升級專家** - 升級指定數量的塔

完成任務可獲得豐厚的金錢獎勵！

### 隨機事件
點擊"提前開始"按鈕時，有30%機率觸發特殊事件：
- **精英入侵** - 挑戰精英戰士
- **巨型威脅** - 對抗高血量怪物
- **速度惡魔** - 面對超速刺客

成功擊敗特殊敵人可獲得額外金錢和特殊獎勵！

### 升級系統
- 每座塔可升級2-3次（機槍塔可升4級）
- 升級提升傷害、射程、射速
- 升級費用遞增（每次×1.5）
- 3級時解鎖特殊能力

## 🚀 快速開始

1. 克隆或下載此倉庫
2. 打開 `index.html` 在瀏覽器中運行
3. 開始遊戲！

## 🎮 遊戲玩法

1. **建造防禦塔**：點擊左側的塔按鈕，然後在右側遊戲區域點擊放置
2. **升級塔**：選擇已建造的塔，點擊"升級"按鈕
3. **出售塔**：選擇塔後點擊"出售"按鈕（獲得50%退款）
4. **開始波次**：等待自動開始或點擊"提前開始"獲得額外50金
5. **完成任務**：完成左側任務面板中的任務獲得獎勵
